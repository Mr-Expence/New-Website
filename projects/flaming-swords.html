Makes the Fire Aspect Enchantment show flames





View on Modrinth Download View on CurseForge





This Resource pack changes the texture for swords when you enchant it with Fire Aspect.

It needs a cit mod for 1.21.4 or below as 1.21.5 adds custom item textures.

There are diffrent colours for the Copper and Netherite swords

3D Mace and Bow from tixul's Enchantment Outlines

Flaming Swords now has support for tixul's Enchantment Outlines which adds an animated glowing outline to enchanted items.

The flames will glow brighter and animate when the item is enchanted with Fire Aspect.

You can toggle the glowing using ResPackOpts

If there is no animations try turning off `Animate Only Visible Textures` in the Video Settings
